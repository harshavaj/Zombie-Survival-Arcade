## Gameplay

- **Objective:** Survive as long as possible against increasing waves of zombies.  
- **Controls:**
  - `W A S D` → Move the player  
  - Mouse → Aim  
  - Left Click → Shoot  
  - `Enter` → Start / Restart the game  

- **Features:**
  - Player sprite rotation toward mouse  
  - Bullet and zombie sprite interactions  
  - Health tracking  
  - Score tracking  
  - Dynamic difficulty scaling (zombie speed & spawn rate increase per wave)  
  - Start menu & game over screen
